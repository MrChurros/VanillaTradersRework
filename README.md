# Vanilla Traders Rework

![Vanilla Traders Rework](https://i.imgur.com/vO29KCJ.png)

## Overview

**Vanilla Traders Rework** is a visual enhancement pack that reimagines the original Escape from Tarkov traders with a more realistic, immersive, and cohesive appearance.

Each portrait has been carefully redesigned to preserve the identity, personality, and lore of the original characters while bringing them closer to the grounded and gritty atmosphere that defines Tarkov.

The goal is simple:

> Make every trader feel like they truly belong in the same world.

---

## Features

✔ High-quality realistic trader portraits  
✔ Consistent visual style across all traders  
✔ Lore-friendly character interpretation  
✔ Improved facial detail and realism  
✔ Better visual cohesion between trader screens  
✔ Designed to fit naturally within Tarkov's UI  
✔ Preserves the identity of the original traders  
✔ No gameplay changes

---

## Design Philosophy

Rather than reinventing the traders, this project focuses on refining them.

Every portrait was created with attention to:

- Character background and role
- Existing Tarkov lore
- Realistic proportions and clothing
- Consistent lighting and color grading
- Visual harmony across the entire trader roster

The result is a set of portraits that feel authentic to Escape from Tarkov while offering a more modern and immersive presentation.

## Compatibility

- Compatible with all SPT 4.0.x versions
- Portrait replacement only
- No gameplay changes
- Safe to add or remove from existing profiles
- Works with existing trader progression

## Installation

1. Download the latest release.
2. Extract the archive.
3. Copy the files to your SPT installation.
4. Launch the game.
